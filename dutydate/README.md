# ระบบเวร – MySQL Setup Guide

ระบบนี้ถูกปรับเปลี่ยนให้ใช้ฐานข้อมูล **MySQL** ภายในโปรเจคของคุณเอง ผ่านไฟล์ `connect_db.php` 

## 📋 1. สร้างตารางในฐานข้อมูล MySQL

เนื่องจากคุณเชื่อมต่อกับฐานข้อมูลชื่อ `db_multimenu` ให้ทำการรันคำสั่ง SQL สร้างตาราง `duty` ดังนี้ (สามารถนำไปรันใน phpMyAdmin ได้เลย):

```sql
CREATE TABLE `duty` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `date` varchar(20) NOT NULL,
  `shift` varchar(20) NOT NULL,
  `slot` int(11) NOT NULL,
  `name` varchar(100) NOT NULL,
  PRIMARY KEY (`id`),
  UNIQUE KEY `duty_date_shift_slot_idx` (`date`,`shift`,`slot`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

**หมายเหตุ:** การตั้งค่า `UNIQUE KEY` จำเป็นมากเพื่อให้ระบบทำงานแก้ไขหรือลบล้างข้อมูลในแต่ละช่องเวรได้อย่างถูกต้อง

## 🔧 2. การเชื่อมต่อฐานข้อมูล

ระบบดึงข้อมูลการเชื่อมต่อ MySQL จากไฟล์ของคุณที่:
`../config/connect_db.php`

หากหน้าเว็บแสดง Error การเชื่อมต่อฐานข้อมูล ให้ตรวจสอบไฟล์ `connect_db.php` หรือพาธของไฟล์ใหม่อีกครั้ง

## 🚀 3. การใช้งาน

ระบบพร้อมใช้งานทันที ไม่มีต้องตั้งค่า API ภายนอกเพิ่มเติม:
- เปิด `http://localhost/newmix/dutydate/index.php`
- ข้อมูลทั้งหมดจะถูก อ่าน/เขียน ตรงเข้าสู่ MySQL ผ่านไฟล์ `api.php` ทันที

| การกระทำ | ผลลัพธ์ |
|---|---|
| คลิกแถบ 🌅 เช้า / ☀️ บ่าย / 🌙 ดึก | เปิด modal เลือกชื่อ |
| เลือกชื่อ | บันทึกลง MySQL ทันที |
| กด "ล้างออก" | ลบเวรออกช่องนั้นจากตาราง |
| กดลิงก์ชื่อ สีๆ ด้านบน | เปิดดูตารางเวรเฉพาะของตัวเองได้เลย |
